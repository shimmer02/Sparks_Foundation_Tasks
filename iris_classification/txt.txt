IRIS CLASSIFIER USING LOGISTIC REGRESSION:
created by:-  Suvam Jana

Machine Learning App to find out iris flower class from an input of petal and sepal dimensions. The model uses superviesd learning method to train on the given dataset to predict the output with accuracies upto 97 percent on test dataset.

How to use:
1. Import all files from repository
2. Assuming all dependencies are installed, open app.py and run in terminal 'streamlit run app.py'

Tools used:
1. Pandas
2. NumPy
3. Sklearn(Logistic Regression)
